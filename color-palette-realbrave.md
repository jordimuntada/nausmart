# RealBrave Website Color Palette

## Primary Brand Colors

### Main Brand Colors
- **Primary Blue**: `#1E40AF` (RGB: 30, 64, 175)
  - Used for: Logo, buttons, links, accent elements
  - Tailwind class: `primary`

- **Secondary Green**: `#10B981` (RGB: 16, 185, 129)
  - Used for: Logo gradients, secondary accents
  - Tailwind class: `secondary`

### Logo Specific Colors
- **Dark Navy**: `#0F172A` (RGB: 15, 23, 42)
  - Used for: Logo text gradient, roof elements
  - Creates contrast and depth in the logo

- **White**: `#FFFFFF` (RGB: 255, 255, 255)
  - Used for: Logo building details, navigation background, card backgrounds
  - Provides clean, professional contrast

## Gradient Combinations

### Logo Gradient
- **Primary Gradient**: `#1E40AF` → `#10B981`
  - Creates the main building structure in the logo
  - Used for brand elements and hero sections

### Text Gradient
- **Text Gradient**: `#1E40AF` → `#0F172A`
  - Used for logo typography
  - Creates depth and professional appearance

## Usage Guidelines

### Primary Usage
- Use `#1E40AF` (Primary Blue) for main CTAs, links, and brand elements
- Use `#10B981` (Secondary Green) for accents and success states
- Combine both in gradients for hero sections and brand elements

### Background Usage
- White (`#FFFFFF`) for main content areas
- Gray-50 (`#F9FAFB`) for section backgrounds
- Gray-900 (`#111827`) for footer and dark sections

### Text Usage
- Gray-900 for headings and important text
- Gray-600 for body text
- Gray-400 for muted/secondary text
- White for text on dark backgrounds

### Status Indicators
- Green variants for available/success states
- Orange variants for pending/warning states
- Red variants for unavailable/error states

## Accessibility Notes
- All color combinations meet WCAG 2.1 AA contrast requirements
- Primary blue provides sufficient contrast on white backgrounds
- Gray text colors ensure readability across all use cases
- Status colors are distinguishable for colorblind users
